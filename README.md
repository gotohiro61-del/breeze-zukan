# BREEZE 海辺の王国 ― 体験ずかん（プロトタイプ）

ブリーズ（手ぶらでキャンプできる、家族特化の海辺キャンプ場）の「すべての体験」を、
ルネッサンス沖縄のように **写真カード → クリックで詳細** が出る "図鑑型コンテンツ" として
見せるための **動くイメージ（プロトタイプ）** です。担当者とのイメージ共有用です。

---

## 開き方

### ✅ いちばん簡単・共有もこれ：`BREEZE-zukan.html`（単体ファイル）
**`BREEZE-zukan.html` をダブルクリック**するだけ。画像も全部この1ファイルに埋め込み済みなので、
**フォルダもサーバーもネット接続も不要**。担当者に渡すときは **このファイル1つをメール添付**すればOK
（相手もダブルクリックで見られます）。※約5MB。

### 編集用：`index.html`＋`images/`
中身を編集するときはこちら（`index.html` と `images/` フォルダはセット）。
編集後に `python build_standalone.py` を実行すると、最新内容で `BREEZE-zukan.html` を作り直せます。

## このプロトタイプで体感できること

- 🌊 **あそびの種類**（海あそび／冒険ひろば／みんなあそび／つくる・まなぶ／たき火・ごはん／大人のリラックス）と
  👨‍👩‍👧 **だれと**（幼児〜／小学生〜／親子いっしょ／大人だけでも）で **絞り込み**
- 🖼 カードは **写真＋やること（ひとこと）** だけのシンプル表示。クリックすると **「👑 王国ストーリー」＋基本情報** が開く
- 👑 すべての体験を **海辺の王国・子供王国の物語** として紹介（読むと「楽しそう！」が伝わるように）。詳細では大人にうれしいポイントも一言添えています
- 🏕 **予約導線**：各体験の詳細に「ご予約・空き状況を見る」ボタン（現在は breeze-camp.com に接続。実際の予約ページへ差し替え可）
- 👑 **制覇スタンプ（図鑑のゲーム化）**：体験を集めると上部バーが伸びる。リピート動機づけの仕掛け（記録は端末に保存）
- 💡 「**これから**」バッジ＝**構想中サンプル**（王国クリーン隊・無限砂場・夕日のヨギボー・ビーチパラソル・遠投フィッシング・星空望遠鏡）。
  「**構想中も表示する**」のチェックを外すと、**お客様向け（今あるものだけ）** の表示に切り替わります
- 🏷 **料金タグ**（無料／宿泊に含む／レンタル／有料）と **★人気**（サンプル）で一覧の判断をしやすく
- ♿ キーボード操作・読み上げ対応、SEO向けメタ情報も付与
- ➕ 末尾の「あたらしい体験を追加」カード → **現場でコンテンツを増やす・減らす** 運用イメージ

## 現状分析の要点（提案の背景）

| | ルネッサンス沖縄（参考） | ブリーズ現状 |
|---|---|---|
| 見せ方 | アクティビティを1枚ずつ**カード化** | 各ページの説明文・画像内に**散在** |
| 絞り込み | ジャンル／所要時間で**フィルター** | なし |
| 詳細 | カード→**詳細ページ**（動画も） | 図鑑型の機能は**なし** |
| 素材の量 | 充実 | **充実（むしろ豊富）** |

➡ **結論：ブリーズには"素材"は十分あるのに、図鑑型に整理する"機能"がない。**
「すべてをコンテンツ化する」方向性は的確で、現場のコンテンツ意識・リピート満足に直結します。

## 注意（プロトタイプの位置づけ）

- 写真は各体験の**ストーリーの情景に合わせて**選定済み（焚き火＝夜の炎／星空望遠鏡＝天の川／夕日のヨギボー＝夕日の海／テント＝BREEZEのグランピング夜景 など）。
- 写真は **breeze-camp.com の実写** が中心です。右下に「**イメージ**」と表示された9枚（焚き火・駄菓子屋・縄跳び/バドミントン＋構想中6件）は、内容を伝えるための**参考画像（フリー素材＝Wikimedia Commons）＝差し替え用**です。現場写真ができたら `images/` の同名ファイルを置き換えるだけで反映されます。
- 掲載アクティビティは現在 **28件**（今あるもの＋「💡 これから（構想中）」サンプル）。内容・分類・文言は **たたき台** です。
- コンテンツは `index.html` 内の `ITEMS` 配列で管理しています。
  項目を **足す／消す／並べ替える** だけで図鑑が更新される、データ駆動の設計です。
